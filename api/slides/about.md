# About this session

#### What this session covers

+ React.js core library
+ JSX
+ ES2015 (as needed)
+ Testing with Mocha and Enzyme

#### Not covered - but often present in React apps

+ Redux
+ React Router
+ Ajax (axios, fetch, etc.)
+ Promises, Sagas, async/await
+ Node.js + NPM
+ Babel
+ Webpack
+ SASS or LESS
+ Mocha (used in testing example)
